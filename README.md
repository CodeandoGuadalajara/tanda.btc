# ⚡ Tanda BTC

*Tanda BTC* es un sistema de ahorro y crédito rotativo (*ROSCA*) basado en Bitcoin y la Lightning Network. Un grupo de participantes aporta una cantidad fija de BTC en cada ciclo, y uno de ellos recibe el monto total acumulado.

## 🚀 Características  
- *Auto custodiado*: Los fondos permanecen bajo control de los usuarios con su propia semilla.  
- *Rápido y de bajo costo*: Utiliza la Lightning Network para transacciones instantáneas y con bajas comisiones.  
- *Transparente y seguro*: Contratos inteligentes gestionan los pagos y previenen fraudes.  

## 🔧 ¿Cómo funciona?  
1. Un grupo de participantes acuerda una cantidad fija y una frecuencia de contribución.  
2. En cada ciclo, todos los miembros depositan su parte de BTC en un contrato inteligente.  
3. Un participante recibe el monto total acumulado.  
4. El proceso se repite hasta que todos hayan recibido su turno.  

## 🌍 Casos de uso  
- Ahorros colaborativos sin intermediarios.  
- Financiamiento comunitario entre amigos o negocios.  
- Una alternativa descentralizada a las tandas tradicionales.
